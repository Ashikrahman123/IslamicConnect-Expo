# IslamicConnect-Expo

## Project Description
IslamicConnect-Expo is a mobile application built with Expo and React Native that provides Islamic content, community features, and tools to help users learn and practice Islam. The app focuses on accessibility, offline support, and a simple, modern user experience.

## Features
- Quran reader with translations and audio playback
- Daily hadith and reminders
- Prayer times with notifications and Qibla compass
- Community posts and user profiles
- Bookmarking and favorites
- Offline content caching

## Tech Stack
- React Native (Expo)
- JavaScript / TypeScript
- Backend: Node.js, Express (optional)
- Database: MongoDB or Firebase
- Authentication: JWT or Firebase Auth

## Setup Instructions (Development)
1. Clone the repository:
   git clone https://github.com/Ashikrahman123/IslamicConnect-Expo.git
2. Change into the project directory:
   cd IslamicConnect-Expo
3. Install dependencies:
   npm install
   or
   yarn install
4. Install Expo CLI globally if you don't have it:
   npm install -g expo-cli
5. Start the development server:
   expo start
6. Open the project in the Expo Go app (iOS/Android) or an emulator.

## Environment
- Create a `.env` file with any required API keys and configuration (e.g., Firebase config, Quran API keys, etc.).

## Usage
- Run the app via Expo and test features on a physical device or simulator.
- Use provided screens to explore Quran reading, prayer times, and community features.
